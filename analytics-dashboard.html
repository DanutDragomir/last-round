🔧 Setup: Enable Analytics in Production

  1. Create a Google Analytics 4 property at analytics.google.com
  2. Get your Measurement ID (format: G-XXXXXXXXXX)
  3. Update www/analytics.js line 8: const GA_ID = 'G-YOUR-ID';
  4. Deploy and monitor events in Analytics dashboard
  5. Set up alerts for key metrics (session duration, win rate trends)
